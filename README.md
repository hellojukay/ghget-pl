# ghget-pl
高速从 github 下载文件

![demo](demo.gif)

# 安装
## Max OS 和 Linux 
```bash
sudo curl https://raw.githubusercontent.com/hellojukay/ghget-pl/master/ghget.pl -o /bin/ghget && sudo chmod +x /bin/ghget
```
## Windows
```bat
ghget.exe 加入到 PATH
```
# 使用方式
```bash
usage of ghget:
    ghget <httpurl> 下载链接指向的文件

感谢 https://g.ioiox.com/ 提供的免费服务
```
