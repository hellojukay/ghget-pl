# ghget-pl
高速从 github 下载文件

![demo](demo.gif)

# Install
## Max OS and Linux 
```bash
curl https://github.com/hellojukay/ghget-pl/blob/master/ghget.pl | sudo tee /bin/ghget && chmod +x /bin/ghget
```

# 使用方式
```bash
usage of ghget:
    ghget <httpurl> 下载连接执行指向的文件

感谢 https://g.ioiox.com/ 提供的免费服务
```
